# EXCEL SHORTCUTS

Here are some popular excel shortcuts I learnt The Incubator Hub, All thanks to the Sponsor (Mummy G O), The Incubator Hub and all our facilitators.
* Ctrl + 1 (Format Cell)
* Ctrl + ; (Current Date)
* Ctrl + Shift + 3 (Change Date Format)
* Ctrl + F2  (Print Preview)
* Ctrl + F1 (Display or Hide Ribbon)
* Alt + ; (Select Visible cells)
* Ctrl + Shift + L (Apply Filter)
* Ctrl + 0 (Hide Column)
* Ctrl + 9 (Hide Row)
* Ctrl + - (Delete Row or Column)
* F7 (Spell Check)
* Ctrl + T (Create Table)
* Ctrl + Space (Select Entire Column)
* Shift + Space (Select Entire Row)
* Alt + H + A + R (Align Right)
* Alt + H + A + L (Align Left)
* Alt + E + S + V (Paste as Values)
* Alt + + (Auto Sum)
* Ctrl + E (Flash Fill)
